# Семестровый проект по машинному обучению

Выполнили студенты группы э541андэк: Маринкин Олег, Петрова Александра, Чиркизьянова Виктория

Задача взята с ODS: https://ods.ai/competitions/data-fusion2026-cybershelf

Решение задачи мультилейбл классификации: предсказание 41 бинарного таргета для банковских клиентов.
Метрика: **MAP (Mean Average Precision)**.

## Структура

| Файл | Описание |
|---|---|
| `01_EDA.ipynb` | Разведочный анализ: пропуски, распределения, whale-эффекты, корреляции таргетов |
| `02_feature_engineering.ipynb` | Подготовка трёх датасетов: tree_nan, tree_zero, linear |
| `03_models.ipynb` | Подбор гиперпараметров (Optuna), финальное обучение, сабмит |

## Датасеты

- **tree_nan** — числовые фичи с NaN as-is + категориальные. Для LightGBM, CatBoost.
- **tree_zero** — NaN → 0 + `_isnan` флаги. Для XGBoost.
- **linear** — QT-скейлинг + OHE + `_isnan` флаги + chi²-взаимодействия. Для линейных моделей.

## Модели

LightGBM · CatBoost · XGBoost · Logistic Regression
